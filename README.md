
# Projeto de Dinâmica de Voo

Código python para a cadeira de Dinâmica de Voo


## Bibliotecas Necessárias

Correr o seguinte código num terminal

```bash
  pip install numpy
  pip install seaborn
  pip install matplotlib
```
